## Acerca del Proyecto :

El Proyecto web se realizo en laravel - js - blade , este proyecto cumple las sisguientes funciones:

- Sistema de Autentificacion.
- Sistema de roles y permisos.
- Manejo de datos de area de venta.
- Manejo de datos de area de pedidos al proveedor.
- Manejo de datos de area de alamcen.
- generar reporte general de todas las ventas.
- generar boleta de compra.
